Security Policy

Important: This security policy outlines our commitment to protecting our website and users from security threats.

1. Responsible Disclosure

We take security seriously and welcome responsible disclosure of security vulnerabilities. If you discover a security issue, please report it to us immediately.

Security Contact:
Email: ilyushina.law@gmail.com
Response Time: Within 48 hours
Preferred Language: Greek, English, Russian

2. Security Measures

  • HTTPS Only: All connections are encrypted using TLS 1.3
  • Security Headers: Comprehensive security headers to prevent common attacks
  • Content Security Policy: Restricts resource loading to trusted sources
  • XSS Protection: Built-in protection against cross-site scripting
  • Clickjacking Protection: Prevents site from being embedded in frames
  • MIME Type Sniffing Protection: Prevents MIME type confusion attacks

3. Data Protection

  • No sensitive data is stored on the client side
  • All form submissions are handled securely
  • No cookies are used for tracking
  • Analytics data is anonymized

4. Vulnerability Reporting

When reporting a vulnerability, please include:

  • Description of the vulnerability
  • Steps to reproduce
  • Potential impact
  • Suggested fix (if any)

5. Response Process

  1. We will acknowledge receipt within 48 hours
  2. We will investigate and provide updates
  3. We will fix the issue and notify you
  4. We will credit you in our security acknowledgments (if desired)

6. Out of Scope

  • Social engineering attacks
  • Physical attacks
  • Denial of service attacks
  • Third-party services we use

7. Legal

By reporting vulnerabilities, you agree to:

  • Not exploit the vulnerability for malicious purposes
  • Not share the vulnerability with others until it's fixed
  • Comply with applicable laws
